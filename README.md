## ３つのコース
1. **システムエンジニアリングコース**
- プログラミングを基礎から学ぶ
- 大小様々なシステムを扱うエンジニアを育成する
2. **インターネット&セキュリティコース**
- 安全のためのネットワーク技術やセキュティ対策を学ぶ
3. **知能情報コース**
- 生活の一部となった人工知能を基礎から応用まで学ぶ
- 情報社会の未来を作り出す専門家の育成を目指す
